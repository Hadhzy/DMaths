## Function composition
$
f \circ g (x) = f(g(x))
$

If x is not in the domain of g, if must not be in the
domain of f(g(x)).

Any x for which g(x) is not in the domain of g ust not be in the domain of f(g(x)).

Example:
- $f(x) = 2x - 3$
- $g(x) = 5x^2 -1$

$f \circ g (x) = f(g(x))$
$2(5x^2-1)-3 = 10x^2 -5$
